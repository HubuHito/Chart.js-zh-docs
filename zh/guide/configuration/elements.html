可以通过 all datasets the same way 配置所有数据集的图形元素（elements）样式，不用每个数据集单独配置，如一个常见的例子：柱形图描边一样，但是每个柱子的填充色不一样。有四种图形元素可供配置：arc， lines， points 和 bars，配置后，所有改类型的图形元素都会应用，除非被数据集里面的配置覆盖。

# 全局配置

图形元素elements支持全局配置和图表配置，全局配置通过Chart.defaults.elements设置，如下，设置所有柱形图的边框为1px：

提示

全局配置一般在Chart.defaults命名空间下，图表配置一般在options配置项下面，数据集datasets的配置可覆盖全局和图表的配置。

Chart.defaults.elements.bar.borderWidth = 2;

# 点配置

点元素一般用来绘制折线图、雷达图或者气泡图中的点，点元素全局配置项为 Chart.defaults.elements.point。

名称 类型 默认值 描述
radius number 3 半径
pointStyle string|Image 'circle' 点样式
rotation number 0 旋转角度
backgroundColor 颜色 Chart.defaults.backgroundColor 填充色
borderWidth number 1 描边
borderColor 颜色 Chart.defaults.borderColor 描边颜色
hitRadius number 1 Extra radius added to point radius for hit detection.
hoverRadius number 4 悬浮时点半径
hoverBorderWidth number 1 悬浮时描边

# 点样式

可选值如下：

  • 'circle'
  • 'cross'
  • 'crossRot'
  • 'dash'
  • 'line'
  • 'rect'
  • 'rectRounded'
  • 'rectRot'
  • 'star'
  • 'triangle'

如果值类型是图像，则会调用canvasdrawImage (opens new window)方法。

# 线配置

线元素表示绘制折线，全局线元素配置Chart.defaults.elements.line

名称 类型 默认值 描述
tension number 0 张力 (0 表示不是贝塞尔曲线，即直线)
backgroundColor 颜色 Chart.defaults.backgroundColor 填充色
borderWidth number 3 描边
borderColor 颜色 Chart.defaults.borderColor 描边色
borderCapStyle string 'butt' 线帽样式，查看MDN (opens new window)
borderDash number[] [] 虚线模式，查看MDN (opens new window)
borderDashOffset number 0.0 虚线偏移量，查看MDN (opens new window).
borderJoinStyle string 'miter' 连接方式，查看MDN (opens new window)
capBezierPoints boolean true true表示在图表内受贝塞尔曲线控制，false为没限制
cubicInterpolationMode string 'default' 插值模式，查看
fill boolean|string false 如何填充折线以下部分，查看面积图
stepped boolean false true 表示显示阶梯线 (tension 属性会被忽略)

# 条形配置

条形元素用于绘制柱形图的条形，全局配置在Chart.defaults.elements.bar中设置。

名称 类型 默认值 描述
backgroundColor 颜色 Chart.defaults.backgroundColor 填充色
borderWidth number 0 描边
borderColor 颜色 Chart.defaults.borderColor 描边色
borderSkipped string 'start' Skipped (excluded) border: 'start', 'end', 'bottom', 'left', 'top' or 'right'.

# 圆弧配置

圆弧一般用于绘制南丁格尔玫瑰图、圆环图和饼图，全局配置在Chart.defaults.elements.arc中设置。

名称 类型 默认值 描述
angle - for polar only number circumference / (arc count) 弧度
backgroundColor 颜色 Chart.defaults.backgroundColor 填充色
borderAlign string 'center' 对齐方式
borderColor 颜色 '#fff' 描边色
borderWidth number 2 描边
